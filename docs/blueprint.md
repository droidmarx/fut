# **App Name**: FieldSidekick

## Core Features:

- Number Reservation: Allow users to select a number from either side (11 on each side) of the field.
- QR Code Generation: Generate a QR code for payment upon number selection.  Include a tool to ensure the requested user is correctly linked with the generated payment data.
- Payment Confirmation & Number Reservation: After payment confirmation, reserve the number with the user's name.
- Admin Dashboard: Provide an admin dashboard with login (Admin: Elvis@123) to release numbers and edit reservation names.
- Payment Settings: Admin option to configure Pix key and WhatsApp number for payment confirmation sending.  Include a tool to check and confirm the user-provided whatsapp number and PIX key is active.
- User Notifications: Configure Whatsapp message sending upon reservation and confirmation.

## Style Guidelines:

- Primary color: A vibrant blue (#29ABE2) to represent trust and energy associated with sports.
- Background color: Light gray (#F5F5F5) for a clean and modern look.
- Accent color: A bright green (#90EE90) to indicate availability and action.
- Font: 'Inter' sans-serif for a modern, clean and readable interface. Suitable for both headlines and body text.
- Responsive layout to ensure seamless access across devices.
- Use of clear and intuitive icons for easy navigation and user understanding.
- Subtle animations and transitions to provide feedback and enhance user experience.
